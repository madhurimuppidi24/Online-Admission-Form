# Online Admission Form

A simple front-end project built using **HTML** that allows students to fill out an online admission application form with personal, academic, and address details.

## Features
- Personal and academic information fields
- Course selection dropdown
- District and address details
- Declaration checkbox
- Submit button with success page redirect

## Technologies Used
- HTML5

## How to Run
1. Download or clone the repository  
2. Open `index.html` in any web browser  
3. Fill the form and click **Submit**
